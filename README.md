# n8n-lead-management-system
Simple Lead Management System using N8N

Workflow Architecture 
Form → Webhook → Google Sheets → Notification 
Key Features 
✓ Automatic Lead Registration 
✓ Sequential Lead ID Generation 
✓ Real-Time Notifications 
✓ No Manual Data Entry 
Results 
• Eliminated manual data entry. 
• Reduced response time to new leads. 
• Improved lead tracking and organization.
